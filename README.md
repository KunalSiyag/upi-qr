# UPI QR

SEO-first UPI QR code generator and future payment-material builder for Indian businesses, events, and donation campaigns.

## Stack

- Astro
- Tailwind CSS
- React islands
- Cloudflare Pages ready static output

## Start

```bash
npm install
npm run dev
```

## V1 scope

- Homepage targeting high-intent UPI QR keywords
- Browser-side UPI QR generator
- Download PNG
- Initial SEO landing pages for payment-app and donation intent

## Next product layers

- Poster templates
- Branded QR customization
- Printable standees and table cards
- Local-language landing pages
